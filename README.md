# Prompt Chain Tool

A focused tool for managing humor flavors and their associated image and step data. A narrower, more specialized slice of a larger content admin platform.

##  Tech Stack
- **Framework:** Next.js 16 (App Router) + React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Backend:** Supabase (SSR + JS client)
- **Theming:** next-themes
- **Testing:** Jest
